Improve passcode key discovery by retaining weave auth traits in v2 parsing.

## Highlights
- Preserve `weave.trait.auth.*` descriptors in `_parse_v2_inner` so non-`ApplicationKeysTrait` auth payloads can reach key-candidate extraction.
- Prefer auth descriptors during v2 patch `Any` selection to reduce dropped auth blobs in mixed payloads.
- Tighten lock-device marker detection to avoid creating non-lock entities from thermostat/annotation payloads.
