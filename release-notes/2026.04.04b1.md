Fix native Nest coexistence by using the runtime google.rpc status proto.

## Highlights
- Route gateway v1 status imports to the runtime-installed `google.rpc.status_pb2` module instead of the vendored `zzzgoogle` copy.
- Restore the prerelease helper and template so HACS beta builds can be cut from dedicated RC branches without ad hoc release steps.
- Document that `2026.03.28` fixed shared Nest/Weave descriptor reuse, while native Nest coexistence needs this prerelease hotfix.

## Validation
- `python3 -m unittest tests.test_protobuf_compat`
- `python3 -m compileall custom_components/nest_yale_lock tests`

## Breaking changes
- None.
