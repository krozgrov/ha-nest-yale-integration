Fix native Nest coexistence in stable release and remove unsupported guest passcode claims from the README.

## Highlights
- Promote the native Home Assistant Nest coexistence fix from `2026.04.04b1` into the stable `2026.04.06` release.
- Gateway v1 protobufs now import the runtime `google.rpc.status_pb2` module so Home Assistant core and this integration share one `google/rpc/status.proto` descriptor pool.
- Remove unsupported guest passcode feature claims from `README.md` until that workflow is validated end-to-end.
